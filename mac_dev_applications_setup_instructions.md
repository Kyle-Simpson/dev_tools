## Directory structure
Everything in relation to the base directory `/Users/<SOME_USER_ID>` as `~/`

Git repos: `mkdir ~/repos`

Scripts and configurations:
> - ~/.zshrc
> - ~/build_new_env.sh
> - ~/hey_friend.txt
> - etc...
